# 💼 Jonathan Rojas | Portafolio Profesional

Bienvenido a mi portafolio profesional de desarrollo web.

Este proyecto reúne mi trabajo como desarrollador web y diseñador, donde aplico buenas prácticas de programación, diseño responsivo, accesibilidad y organización de código para crear sitios modernos, rápidos y escalables.

---

## 🚀 Acerca del proyecto

Este portafolio fue desarrollado con el objetivo de presentar mis habilidades, proyectos y experiencia profesional mediante una interfaz moderna y una arquitectura organizada.

Actualmente el proyecto se encuentra en desarrollo continuo y se actualiza constantemente con nuevas funcionalidades y mejoras.

---

## ✨ Características

* Diseño moderno y profesional.
* Interfaz totalmente responsive.
* Navegación adaptable para dispositivos móviles.
* Arquitectura modular con PHP.
* Organización del código mediante componentes reutilizables.
* CSS estructurado por módulos.
* Optimización de imágenes.
* Buenas prácticas de accesibilidad.
* Código limpio y mantenible.

## 💻 Instalación

Clona el repositorio:

```bash
git clone https://github.com/JRfreelancer/jonathan-rojas-portfolio.git
```

Entra al proyecto:

```bash
cd jonathan-rojas-portfolio
```

Coloca la carpeta dentro del servidor local (Laragon, XAMPP o similar) y accede desde tu navegador.
